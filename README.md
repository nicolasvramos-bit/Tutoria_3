1. Qual é o problema identificado?
O problema é a falta de controle financeiro, que pode fazer uma pessoa gastar mais dinheiro do que ganha sem perceber.

2. Onde esse problema acontece?
Esse problema acontece no dia a dia, principalmente nas finanças pessoais, familiares ou de pequenos negócios.

3. Quem é afetado por ele?
São afetadas pessoas que precisam administrar seu dinheiro, controlar gastos e acompanhar seus ganhos.

4. Como um programa de computador poderia ajudar a resolver ou minimizar esse problema?
O programa ajuda registrando os ganhos e os gastos, calculando o saldo automaticamente e mostrando a situação financeira do usuário. Assim, fica mais fácil acompanhar as finanças e evitar prejuízos causados por gastos excessivos.
